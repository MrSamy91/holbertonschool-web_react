# React Hooks

Modernizing a React dashboard from class components to functional components using React hooks.

## Learning Objectives

- Understanding the transition from class to functional components
- Learning different React hooks (useState, useContext, useEffect…)
- Understanding component optimization with React.memo
- Building custom React hooks
- Developing testing strategies for hooks
- Understanding the reducer pattern before Redux

## Requirements

- Ubuntu 20.04 LTS, Node 20.x.x, npm 10.x.x
- Jest 29.7.0 for testing
- Import React only in main.jsx
- Files end with a new line
